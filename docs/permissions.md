# Permissions

Use permissions to control access to EzChat moderation and logging tools.

## Core Staff Permissions

- `ezchat.logs` — Access EzChat log viewing commands.
- `ezchat.mute` — Use permanent mute command.
- `ezchat.mutetemp` — Use temporary mute command.

Assign these permissions through your permissions plugin (for example, LuckPerms) to the appropriate staff roles.
