# Installation

Follow these steps to install EzChat on your server:

1. Download the latest `EzChat` plugin JAR file.
2. Stop your Minecraft server (recommended).
3. Place the JAR file in your server's `plugins/` folder.
4. Start or restart the server.
5. Confirm EzChat loads successfully in the console logs.

After startup, EzChat will generate its default configuration files in the plugin directory.
