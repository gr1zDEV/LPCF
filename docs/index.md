# EzChat Documentation

EzChat is a communication management plugin for Minecraft servers, designed to give administrators control over player messaging while keeping chat systems easy to use.

## Main Features

- Global and private communication controls
- In-game mail support for offline messaging
- Logging and audit tools for moderation
- Mute and temporary mute system
- Permission-based access for staff tools

Use this documentation to set up EzChat and learn how to configure commands, permissions, logs, and moderation settings.
