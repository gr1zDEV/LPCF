# Mute System

The mute system allows staff to limit player communication when rules are violated.

## Commands

- `/ezchat mute <player> <reason>` — Apply a permanent mute.
- `/ezchat mutetemp <player> <duration> <reason>` — Apply a temporary mute.

## Mute Effects

When a player is muted, communication actions may be blocked, including:

- Public chat messages
- Private messages
- Mail usage (depending on configuration)

Muted players should receive feedback when they attempt restricted actions.
