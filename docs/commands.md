# Commands

EzChat provides commands for chat control, private messaging, mail, logs, and moderation.

## Player Commands

- `/togglechat` — Toggle global chat visibility for yourself.
- `/togglemsg` — Enable or disable private messages.
- `/togglemail` — Enable or disable mail notifications/messages.
- `/msg <player> <message>` — Send a private message.
- `/reply <message>` — Reply to the last private message.
- `/mail <player> <message>` — Send mail to a player.

## Staff / Admin Commands

- `/ezchat logs` — View communication and audit logs.
- `/ezchat mute <player> <reason>` — Permanently mute a player.
- `/ezchat mutetemp <player> <duration> <reason>` — Temporarily mute a player.

> Note: Exact argument formats may vary by plugin version and command aliases.
