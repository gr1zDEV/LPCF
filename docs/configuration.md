# Configuration

EzChat uses multiple configuration files to separate features clearly.

## Configuration Files

- `config.yml` — Main plugin settings and global behavior.
- `private-message.yml` — Private message formatting and behavior.
- `mail.yml` — Mail feature settings.
- `logs.yml` — Logging configuration and retention behavior.
- `mute.yml` — Mute system settings and defaults.

## Recommended Workflow

1. Stop the server before major configuration changes.
2. Edit YAML files carefully (spaces only, no tabs).
3. Save changes and restart the server.
4. Verify behavior in-game and check console logs for errors.
